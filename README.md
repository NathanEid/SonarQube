# SonarQube

### Lab Day 01

#### Install SonarQube with helm

    git clone https://github.com/kareemelkasaby1/Sonarqube-Demo.git
    helm create postgress
    helm create sonarqube
    helm install postgress ./postgress/
    helm install sonarqube ./sonarqube/
